1.Cos'è ReactJS? (risposta aperta)

React è una libreria JavaScript per la creazione di interfacce utente, Sviluppata nel 2013 all'interno di Facebook, adesso React è una libreria open-source supportata da una grande community di programmatori. React consente di sviluppare applicazioni dinamiche che non necessitano di ricaricare la pagina per visualizzare i dati modificati. Inoltre nelle applicazioni React le modifiche effettuate sul codice si possono visualizzare in tempo reale, permettendo uno sviluppo rapido, efficiente e flessibile delle applicazioni web.

6.Cos'è un componente React?

Un blocco di logica/contenuto riutilizzabile all'interno dell'applicazione

8.Qual è la differenza tra componenti creati come funzione e componenti creati come classe?

I componenti creati come funzione sono più semplici da scrivere ma con meno funzionalità, mentre i componenti creati come classe necessitano in genere di un maggior numero di linee di codice ma possiedono più funzionalità.
